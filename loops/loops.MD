  ### break key Word
  this use to exit from loop.
  ```java
  for(int i=0;i<10;i++){
     if(i==3){
      break;
      }
    System.out.print(i);
  }
System.out.print("WE ARE OUT OF LOOP");

  ```

  ### continue key word
  this is used to skip an iteration
  ```java
    for(int i=0;i<5;i++){
       if(i==3){
        continue;
       }
  System.out.print(i);
    }
  ```
